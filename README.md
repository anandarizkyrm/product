# STEP TO RUN :

- clone this repository
- `yarn install`
- setup you env based on `.env.example`
- start your db
- run migration with the following command `npx sequelize-cli db:migrate`
- run seeder with the following command `npx sequelize-cli db:seed:all`
- `yarn dev`
